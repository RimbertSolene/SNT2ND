Exercice 2

Définir la fonction `min(liste)` pour trouver le plus petit élément :

liste1 = [2, 7, 90, 10, 50, 1]

def max(liste):
    max_value = liste[0]
    for i in liste:
        if i > max_value:
            max_value = i
    return max_value

def min(liste):
    min_value = liste[0]
    for i in liste:
        if i < min_value:
            min_value = i
    return min_value

print("Le plus grand est :", max(liste1))
print("Le plus petit est :", min(liste1))

Exercice 3

Définir la fonction `trouve(liste, nb)` qui cherche une valeur :

def trouve(liste, nb):
    return nb in liste

liste1 = [2, 7, 90, 10, 50, 1]
print(trouve(liste1, 7)) # Affiche True
print(trouve(liste1, 99)) # Affiche False
