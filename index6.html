Python - Cours 6 : Listes et Tableaux

Introduction

Une liste est une structure de données qui contient une série de valeurs.

Exemple :

tab = [2, 3, 7, 10, 22]

Accéder aux éléments d’une liste

On peut appeler les éléments d'une liste par leur position (indice ou index) :

print(tab[0]) # affiche 2
print(tab[3]) # affiche 10

Obtenir la taille d'une liste

print(len(tab)) # affiche 5

Listes avec différents types de valeurs

Une liste peut contenir des valeurs de types différents :

mixte = ["girafe", 5, "souris", 0.15]

Concaténation et répétition

Les listes supportent l'opérateur + pour la concaténation et * pour la répétition :

ani1 = ["girafe", "tigre"]
ani2 = ["singe", "souris"]
print(ani1 + ani2) # affiche ['girafe', 'tigre', 'singe', 'souris']
print(ani1 * 3) # affiche ['girafe', 'tigre', 'girafe', 'tigre', 'girafe', 'tigre']

Création et modification d’une liste

Création d’une liste vide et ajout d’éléments :

a = []
a = a + [15]
print(a) # affiche [15]
a.append(13)
print(a) # affiche [15, 13]
